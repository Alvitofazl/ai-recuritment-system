# AI-Powered Recruitment System

## Overview
This project is an AI-driven recruitment automation system designed to streamline candidate screening, ranking, and shortlisting.

## Problem
Manual candidate screening is time-consuming and inefficient, especially when dealing with large volumes of applications.

## Solution
I built an automated workflow using AI and automation tools to evaluate and rank candidates based on predefined criteria.

## Tools Used
- OpenAI API
- Google Sheets
- Make (Integromat)

## Workflow
1. Candidate data is collected via Google Forms  
2. Data is stored in Google Sheets  
3. Make triggers automation  
4. OpenAI evaluates candidates  
5. System generates ranking & shortlist  

## Impact
- Reduced manual screening effort  
- Improved decision-making consistency  
- Structured candidate evaluation
