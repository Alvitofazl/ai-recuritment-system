# AI-Powered Recruitment System

## 🚀 Overview
This project is an AI-driven recruitment automation system designed to streamline candidate screening, ranking, and shortlisting.

Instead of manually reviewing candidates, this system automates the entire evaluation process using AI and workflow automation tools, enabling faster and more consistent hiring decisions.

---

## ❗ Problem
Manual candidate screening is:
- Time-consuming
- Inconsistent across evaluators
- Difficult to scale with high application volume

---

## 💡 Solution
I built an end-to-end automated recruitment workflow that:
- Processes candidate data automatically
- Uses AI to evaluate candidates based on defined criteria
- Generates structured scoring, ranking, and hiring recommendations

---

## 🛠️ Tools & Technologies
- OpenAI API (AI evaluation)
- Google Sheets (data storage & processing)
- Make / Integromat (workflow automation)

---

## 🔄 Workflow
1. Candidate data is collected and stored in Google Sheets  
2. System identifies unprocessed candidates  
3. Make triggers automation workflow  
4. OpenAI evaluates candidates based on role criteria  
5. Output is parsed into structured data  
6. Results are saved with score, ranking, and status  
7. System marks candidates as processed  

---

## 🏗️ System Architecture
Input → Processing → Output

- **Input:** Candidate data (Google Sheets)
- **Processing:** AI evaluation via OpenAI + automation via Make
- **Output:** Structured candidate ranking, scoring, and status (Shortlist / Review / Drop)

---

## ✨ Key Features
- End-to-end workflow automation (minimal manual intervention)
- AI-based candidate evaluation
- Automated ranking and shortlist generation
- Structured and consistent decision-making system
- Scalable recruitment workflow

---

## 📈 Impact
- Reduced manual screening effort  
- Improved decision-making consistency  
- Enabled structured candidate evaluation  
- Increased efficiency in recruitment workflow
